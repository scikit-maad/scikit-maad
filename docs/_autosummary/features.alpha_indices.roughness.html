features.alpha_indices.roughness¶

features.alpha_indices.roughness(x, norm=None, axis=0)[source]¶

Computes the roughness (depends on the number of peaks and their amplitude) of a vector or matrix x (i.e. waveform, spectrogram…) Roughness = sum(second_derivation(x)²)

Parameters
xndarray of floats

x is a vector (1d) or a matrix (2d)

normboolean, optional. Default is None

  • ‘global’ : normalize by the maximum value in the vector or matrix

  • ‘per_axis’ : normalize by the maximum value found along each axis

axisint, optional, default is 0

select the axis where the second derivation is computed

if x is a vector, axis=0

if x is a 2d ndarray, axis=0 => rows, axis=1 => columns

Returns
yfloat or ndarray of floats

References

Described in [Ramsay JO, Silverman BW (2005) Functional data analysis.] Ported from SEEWAVE R Package
